# Car Voice Assistant Simulator

A futuristic car control simulator that uses voice commands to interact with vehicle components in real-time.

# Features
- Voice command recognition (Web Speech API)
- Real-time car animation (SVG)
- Interactive controls:
  - Window
  - Trunk
  - Lights
  - AC
  - Doors
- HUD-style modern UI

# Tech Stack
- HTML
- CSS (Animations + UI)
- JavaScript (Speech API + Logic)

# How to Run
1. Open index.html in browser (Chrome recommended)
2. Allow microphone access
3. Click "Start Listening"
4. Speak commands like:
   - "open window"
   - "lights on"
   - "lock doors"

# Note
- Works best in Chrome (Web Speech API support required)
